Student Innovation Project (SIP)

AI-Assisted Microsoft Enterprise Cleanup

Most organizations accumulate identity “directory debt” over time: inactive users, orphaned mailboxes, unused groups, and permissions that never got cleaned up. This increases attack surface and makes cleanup risky.

My SIP is an audit-first PowerShell module that generates clear HTML/CSV reports, adds policy-aware AI summaries (optional RAG), and requires explicit approval before any remediation runs. Logs capture before/after state and support rollback for supported actions.
